表格

<th>为表头，会自动加粗居中

属性
属性名 属性值 描述
align left、center、right 对齐方式
border 1 或 “” 是否有边框，默认为””，表示无边框
cellpading 像素值 单元边与内容之间空白，默认1像素
cellspacing 像素值 单元格之间空白，默认2像素
width 像素值或百分比 表格宽度
合并单元格

合并方式：

  • 跨行合并：rowspan=”合并单元格个数”
  • 跨列合并：colspan=”合并单元格个数”

目标单元格：

  • 跨行：最上侧单元格为目标单元格，此处写合并代码
  • 跨列：最左侧单元格为目标单元格，此处写合并代码

列表

无序列表（前面有个点）、有序列表（前面有序号）、自定义列表

无序列表

<ul>内只可放<li>，<li>内客房任意标签

自定义列表
<dl>
    <dt>名词</dt>
    <dd>名词解释1</dd>
    <dd>名词解释2</dd>
</dl>

<dl>内只可放<dt>和<dd>

表单

<form action="url" method="提交方式" name="表单域名称">
  各种元素
</form>
常用属性
属性 属性值 作用
action url地址 把表单数据发送到url
method get/post 设置提交方法
name 名称 表单域名称，用于区分
表单控件
  1. input 输入
  2. select 下拉
  3. textarea 文本域
<input>
<input type="属性值" />
属性值 描述
button 可点击按钮，通过js启动脚本
radio 单选按钮
checkbox 复选框（多选）
file 输入字段和”浏览”按钮，供文件上传
hidden 隐藏的输入字段
image 图像提交按钮
password 密码字段
reset 重置按钮，清楚表单数据
submit 提交按钮，将表单数据发送到服务器
text 输入字段，默认宽度20个字符

type之外的其他属性：

属性 属性值 描述
name 用户自定义 定义元素名称
value 用户自定义 规定元素值
checked checked 规定元素首次加载时被选中
maxlength 正整数 规定输入字段的字符最大长度

一组多个单选按钮name需相同，否则一组中每个单选钮可同时选
